Photo naming convention:

  hero.png       - the big top photo
  1.png          - gallery photo 1
  2.png          - gallery photo 2
  3.png ... N.png - additional gallery photos, numbered in order

To add a new photo:
  1. Drop it into this folder as the next number (e.g. if you have 1-7, save it as 8.png)
  2. Open the site locally with ?edit=1 in the URL
  3. Photos tab → "+ Add gallery photo" (auto-uses the next number)
  4. Copy code → paste into index.html → push

You can delete this README.txt once you've read it.
